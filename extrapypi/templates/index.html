Welcome to extrapypi server

Go to dashboard to see all packages.

Uploading packages

Extrapypi is compliant with setuptools / twine, you just need to update your `.pypirc` like this

[distutils] index-servers = local [local] username=admin password=admin repository=http://mypypiurl.com/simple/

And then you can run

python setup.py bdist_wheel upload -r local

Or using twine

twine upload -r local dist/extra_pypi-0.1-py3.5.egg

Installing packages from extrapypi

Extrapypi is built to be used as an extra index, so you can simply run :

pip install extrapypi --extra-index-url http://user:password@mypypiurl.com/simple/

or directly updating your `pip.conf` file

[global] extra-index-url = http://user:password@mypypiurl.com/simple/
